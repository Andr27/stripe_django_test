# Stripe Django Test

Простой Django-проект с оплатой через Stripe.

## Запуск

```bash
docker compose up --build -d
```

---

## Как пользоваться

### Просмотр товара (Item)

```
http://127.0.0.0:8000/item/1
```

 где `1` — это ID товара

---

### 💳 Создание заказа (Order)

```
http://127.0.0.0:8000/order/1
```

 где `1` — ID товара, для которого создаётся заказ

После перехода откроется страница оплаты Stripe.

---

## ⚠️ Важно

* Убедись, что в базе есть товары (`Item`)
* Если `/item/1` или `/order/1` не работают — значит нет записи с таким ID
